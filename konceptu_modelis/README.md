# PlantUML koncepta modeļa ģenerēšana lokāli

1. Lokālajā IDE (piemēram, VS Code) instalēt PlantUML paplašinājumu (PlantUML by jebbs);
2. Atvērt *.plantuml failu;
3. Veikt nepieciešamās izmaiņas;
4. Lai apskatītu diagrammu, Alt + D PlantUML diagrammas ģenerēšanai (option + D - MacOS);
5. Nepieciešamības gadījumā atjaunot saiti galvenajā README failā.
